# 示例数据源目录

此目录用于存放后续模块二、模块三开发所需的真实或脱敏示例数据源。

## 同步规则

- 本目录下的数据会随 GitHub 同步，用于项目示例、测试和功能验收。
- 请只放置允许公开或团队共享的数据。
- 若包含敏感信息，请先脱敏后再放入此目录。
- 临时文件、个人私有文件或不希望上传的数据，请不要放在此目录，或使用 `.user.csv` / `.user.xlsx` 后缀。

## 建议数据格式

推荐提供 `.xlsx` 或 `.csv`，至少包含：

- 日期列
- 处理方式列
- 一个或多个数值型植株参数列

示例表头：

```text
日期,处理方式,单果重,糖度,酸度,产量
```
