# 科研图表样式规范

本文档记录项目内科研图表的默认样式，用于模块二、模块三及后续图表输出保持一致。

## 通用原则

- 图表优先服务科研数据表达，避免装饰性视觉元素。
- 默认使用灰度风格，确保黑白打印和论文排版中仍可辨认。
- 默认画布比例为 4:3，导出分辨率不低于 300 dpi。
- 图表中的数值展示统一使用三位有效数字。
- 中文标签需要优先使用系统可用中文字体，避免导出图片出现方框或乱码。

## 模块二：日期趋势散点图

- 模块二图表用于展示数据分布和随时间变化的趋势。
- x 轴默认使用模块一生成的 `isoweek`。
- y 轴使用当前选择的参数列。
- 按 `处理方式` 分组绘制原始散点。
- 同一 `isoweek` 内不同处理可在 x 轴方向轻微错开，以减少完全重叠。
- 原始点可添加极小确定性 y jitter，用于显示堆叠密度；均值计算不受 jitter 影响。
- 每个 `处理方式 + isoweek` 计算均值点，并用折线连接同一处理在不同周的均值点。
- 均值点和均值折线必须保持在真实 `isoweek` 坐标上。
- 散点使用白色填充、灰度边框；均值折线使用同组灰度。
- 图例区分各处理的原始点和均值线。

## 模块二：处理分布散点图

- 处理分布图用于展示全周期内不同处理的数据分布。
- x 轴使用 `处理方式`。
- y 轴使用当前选择的参数列。
- 每个处理组内使用确定性横向 jitter 展开原始点，避免大量点压成一列。
- 对同一处理内相近 y 值进行轻量分箱，局部重复越多，点可更大或更深，以表达富集。
- 不绘制柱体，不绘制误差线。
- 不在该图中做显著性标注；显著性表达保留在模块三。

## 模块三：显著性柱状图

- 模块三图表用于展示处理间均值比较和显著性标注。
- 柱体宽度默认使用 `0.4`。
- 两组数据柱状图采用等距布局：
  - x 轴左边界为 `0`。
  - 第一组柱中心为 `1`。
  - 第二组柱中心为 `2`。
  - x 轴右边界为 `3`。
- 多组柱状图应保持柱中心均匀分布，并在两侧保留约一个组间距。
- 柱体默认使用灰度填充、黑色边框，边框线宽约 `0.8`。
- 两组显著性比较默认添加星号标注。
- 多组显著性比较建议添加 Tukey HSD 字母分组。

## y 轴范围

- y 轴一般不强制从 0 开始。
- 模块二趋势图根据清洗后的参数值范围自动留白。
- 模块三柱状图根据 `mean ± error` 的最小值和最大值自动计算上下限。
- 上下限默认在数据范围外保留约 20% 富余。
- 若存在误差线文字或显著性标注，上边界需要额外留白，避免文字被裁切。

## 误差线与标注

- 误差线主要用于模块三柱状图。
- 默认误差线为 SEM。
- 界面或导出逻辑可允许用户切换为 SD。
- 图中误差线含义必须与实际绘制一致：
  - 选择 SEM 时，误差线为 `mean ± SEM`。
  - 选择 SD 时，误差线为 `mean ± SD`。
- 每个柱体的误差线上方标注 `mean ± error`。
- 标注数值使用三位有效数字。
- 标注位置应略高于误差线上端，避免与误差线重叠。

## 数据处理口径

- 模块二和模块三图表均应基于模块二统计口径生成。
- 模块二默认按组内 IQR 1.5 倍识别离群值。
- 统计和图表默认使用剔除离群值后的数据。
- 离群值报告应保留数据 `序号`，用于回溯原始记录。
- 模块三显著性图表应复用模块二清洗后的数据，不重复定义离群值口径。
