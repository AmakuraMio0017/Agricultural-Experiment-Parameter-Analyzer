# 科研图表样式规范

本文档记录项目内科研图表的默认样式，用于模块二、模块三及后续图表输出保持一致。

## 通用原则

- 图表优先服务科研数据表达，避免装饰性视觉元素。
- 默认使用灰度风格，确保黑白打印和论文排版中仍可辨认。
- 默认画布比例为 4:3，导出分辨率不低于 300 dpi。
- 图表中的数值展示统一使用三位有效数字。
- 中文标签需要优先使用系统可用中文字体，避免导出图片出现方框或乱码。

## 柱状图

- 柱体宽度默认使用 `0.4`。
- 两组数据柱状图采用等距布局：
  - x 轴左边界为 `0`。
  - 第一组柱中心为 `1`。
  - 第二组柱中心为 `2`。
  - x 轴右边界为 `3`。
- 多组柱状图应保持柱中心均匀分布，并在两侧保留约一个组间距。
- 柱体默认使用灰度填充、黑色边框，边框线宽约 `0.8`。

## y 轴范围

- y 轴一般不强制从 0 开始。
- y 轴上下限应根据 `mean ± error` 的最小值和最大值自动计算。
- 上下限默认在数据范围外保留约 20% 富余。
- 若存在误差线文字标注，上边界需要额外留白，避免文字被裁切。

## 误差线与标注

- 默认误差线为 SEM。
- 界面或导出逻辑可允许用户切换为 SD。
- 图中误差线含义必须与实际绘制一致：
  - 选择 SEM 时，误差线为 `mean ± SEM`。
  - 选择 SD 时，误差线为 `mean ± SD`。
- 每个柱体的误差线上方标注 `mean ± error`。
- 标注数值使用三位有效数字。
- 标注位置应略高于误差线上端，避免与误差线重叠。

## 数据处理口径

- 图表应基于模块二统计口径生成。
- 模块二默认按组内 IQR 1.5 倍识别离群值。
- 统计和图表默认使用剔除离群值后的数据。
- 离群值报告应保留数据 `序号`，用于回溯原始记录。
- 模块三显著性图表应复用模块二清洗后的数据和图表样式，不重复定义绘图风格。

## 模块三扩展

- 两组显著性比较默认在本样式基础上添加星号标注。
- 多组显著性比较建议在本样式基础上添加 Tukey HSD 字母分组。
- 显著性标注应避免遮挡 `mean ± error` 文本；必要时增加 y 轴上限留白。
